Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b></b>

Nesa frase temos um termo em destaque usando a tag <strong></strong>

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i></i> (não semântica)

Nessa frase temos um termo em Ênfase usando a tag EM (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK

Texto grande e pequeno

Podemos criar textos grandes e textos pequenos nos paragráfos e títulos...

Texto excluído

Podemos marcar um texto como excluído, isso é, que pode ser lido, porém não considerado

Texto inserido

Ou então podemos colocar um texto como inserido, assim mostrando as alterações e o novo contexto

Ele é diferente da tag <u></u> que deixa o texto subinhado (não semântica)

Texto sobrescrito

Para inserir coisas como x2 + 7x + 78 = 0

Texto sobescrito

Para inserir coisas como H2O
